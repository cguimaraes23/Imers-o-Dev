Estou participando da seleção de melhores projetos da imersão Dev com Alura + Google com o Coffee.log! 

Desenvolvi uma aplicação web pensada para resolver um problema comum de quem faz home office ou estuda fora: encontrar o lugar perfeito que una conforto e produtividade. O projeto mapeia as principais cafeterias do Rio de Janeiro, mas vai além do cardápio.

O foco é a infraestrutura para trabalho remoto: o site classifica os locais indicando a disponibilidade de tomadas, a estabilidade do Wi-Fi e o nível de ruído do ambiente. Assim, o usuário consegue filtrar e encontrar seu próximo escritório temporário sem surpresas.
